# Güneş Enerjisi Hasatlı Sensör Ağları (EH-WSN) Performans Analizi

Bu proje, Python ve Pygame kullanılarak geliştirilmiş bir Kablosuz Sensör Ağları (WSN) simülasyon panelidir. Gerçek bir Multi-Hop mimarisinde darboğaz düğümlerini gözlemlemek ve güneş enerjisi hasadının ağ güvenilirliğini nasıl artırdığını test etmek amacıyla tasarlanmıştır.
